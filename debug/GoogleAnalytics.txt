Allow someone to login to GA stats:
	In GA, goto Admin, User Management
	
Set Up Tracking Javascript Code:
	In GA, goto Admin. Under Property, Tracking Info, Tracking Code you will find the javascript to add to the end of index.html just before the closing body tag.
	
Set Up Application Name in the left column of reports (dimension):
	In GA, goto Admin. Under Property, Custom Definitions, Custom Dimensions
	I setup name: "Custom App Name"
	Scope: Hit
	Active: checked
